# Credit Scoring Model - CodeAlpha Internship

**Task 1: Credit Scoring Model**

This project predicts whether a loan applicant is a **good credit risk** (loan will be paid) or **bad credit risk** (loan may default).

### Objective
Predict an individual's creditworthiness using past financial data using classification algorithms as per CodeAlpha Machine Learning Internship Task 1.

### Dataset Used
- **Credit Risk Dataset** (32,000+ records)
- Features: Age, Income, Home Ownership, Employment Length, Loan Amount, Interest Rate, Loan Intent, etc.
- Target: `loan_status` (0 = Good Credit, 1 = Bad Credit)

### Models Trained
- Logistic Regression
- Decision Tree
- Random Forest

### Results
- Best Model: **Random Forest**
- Accuracy: ~93–95% (you can update this with your actual number from Cell 4)
- Evaluated using: Accuracy, Precision, Recall, F1-Score, ROC-AUC

### How to Run
1. Open `Credit_Scoring_Model.ipynb` in Google Colab
2. Run all cells
3. Check the prediction in Cell 5

### Technologies Used
- Python
- Pandas, Scikit-learn, Seaborn, Matplotlib
